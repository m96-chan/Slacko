# Support

## Getting Help

If you encounter issues or have questions about Slacko, here's how to get help:

### Bug Reports

Found a bug? Please [open an issue](https://github.com/m96-chan/Slacko/issues/new) on GitHub with:

- A description of the problem
- Steps to reproduce
- Your OS and Go version (`go version`)

### Feature Requests

Have an idea for a new feature? [Open an issue](https://github.com/m96-chan/Slacko/issues/new) and describe what you'd like to see.

### Common Issues

**"Authentication failed"**
- Make sure you completed the OAuth authorization in your browser
- Try running `slacko` again to re-authorize

**"Not in channel" errors**
- Slacko will automatically join public channels when you send a message
- For private channels, ask a member to invite you

**No messages appearing**
- Check that your Slack App has the required event subscriptions enabled
- Verify Socket Mode is enabled in your Slack App settings
